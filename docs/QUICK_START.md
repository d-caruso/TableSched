# Quick Start

## Frontend

The frontend is an Expo Router app in `frontend/` using React Native Web, Tamagui, React Query, and i18next.

### Run locally

```bash
cd frontend
npm install
npm run web
```

### Required environment variables

Set these in your local `.env` or deployment provider:

- `EXPO_PUBLIC_API_BASE_URL`
- `EXPO_PUBLIC_STRIPE_PUBLISHABLE_KEY`

### Useful scripts

- `npm run web`
- `npm run build`
- `npm run typecheck`
- `npm run lint`
- `npm test`
